# Purpose

This knowledge base stores durable knowledge that should be useful beyond the
current AI session.

Save:

- reusable project decisions and constraints
- commands or workflows that are likely to be reused
- pitfalls and fixes that should not be rediscovered
- source-grounded facts, comparisons, and synthesis

Do not save:

- secrets, credentials, tokens, or private keys
- temporary task progress
- raw logs unless the log itself is durable evidence
- unsupported guesses or unresolved speculation
- content that belongs only in the current project repository
